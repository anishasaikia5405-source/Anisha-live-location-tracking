# Live Location Tracker Pro 🛰️

A cutting-edge, real-time GPS tracking web application with a modern hacker-style dashboard. Track your device's location with precision, monitor speed, battery status, and more.

![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![PRs](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)

## ✨ Features

### 🗺️ Advanced Map System
- Real-time location tracking with Leaflet.js
- Smooth marker animations
- Movement path visualization
- Auto-center functionality
- Multiple zoom levels

### 📊 Live Dashboard
- Current speed display (km/h)
- Battery percentage with charging status
- GPS accuracy indicator
- Total distance traveled
- Live coordinates (DMS format)
- Device information detection

### 🎮 Interactive Controls
- Start/Stop tracking
- Reset movement path
- Export route as JSON
- Sound alert toggle
- GPS status indicators

### 🎨 Modern UI/UX
- Hacker-style dark theme
- Glassmorphism effects
- Animated dashboard cards
- Fully responsive design
- Loading animations
- Mobile-optimized

### 📱 Device Features
- Battery API integration
- Device type detection
- Platform identification
- Vibration support
- Sound notifications

## 🚀 Quick Start

### Option 1: Direct Usage
1. Clone this repository
2. Open `index.html` in a modern browser
3. Allow GPS permissions when prompted
4. Click "Start Tracking" to begin

### Option 2: GitHub Pages
1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Access your live tracker at `https://[username].github.io/[repo-name]`
